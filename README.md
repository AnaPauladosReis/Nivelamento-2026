# 🌱 Joinville Voluntária

**Joinville Voluntária** é um site institucional desenvolvido como projeto acadêmico para o curso de Análise e Desenvolvimento de Sistemas (ADS) do UniSENAI. O objetivo é conectar pessoas interessadas em voluntariado ambiental com projetos ativos na cidade de Joinville/SC, promovendo a sustentabilidade e a participação cidadã.

O site apresenta informações sobre projetos ecológicos, formulários de contato e inscrição, além de uma simulação de login para voluntários. Todo o conteúdo é fictício e voltado para a demonstração de habilidades em desenvolvimento web front-end.

---

## ✨ Funcionalidades

- Página inicial com chamada para ação e vídeo institucional.
- Listagem de **projetos ativos** com cards interativos.
- **Modal de inscrição rápida** para cada projeto (simulação).
- **Formulário de contato** com validação.
- **Página "Sobre"** com a história da organização e equipe fictícia.
- **Simulação de login** (credenciais fixas para demonstração).
- Design responsivo, tipografia personalizada e ícones vetoriais.

---

## 🛠 Tecnologias Utilizadas

- **HTML5** – Estrutura das páginas
- **CSS3** – Estilização customizada (arquivo `style.css`)
- **Bootstrap 5** – Layout responsivo e componentes (navbar, cards, modal)
- **JavaScript (Vanilla)** – Validação de formulários e interatividade (arquivo `script.js`)
- **Font Awesome** – Ícones ilustrativos
- **Google Fonts** – Fonte Poppins
- **YouTube** – Vídeo incorporado na página inicial

## 🚀 Como executar o projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/joinville-voluntaria.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd joinville-voluntaria
   ```
3. Abra o arquivo `index.html` em seu navegador preferido.

Não é necessário instalar dependências ou configurar servidor – o projeto é estático e funciona diretamente no navegador.

## 📌 Observações

- Este é um projeto **acadêmico** e não possui funcionalidades back-end reais.
- As imagens utilizadas são de domínio público ou obtidas de fontes gratuitas (Unsplash, iStock, etc.) apenas para fins educacionais.
- O formulário de contato e inscrição disparam apenas um `alert()` de confirmação – nenhum dado é enviado ou armazenado.
